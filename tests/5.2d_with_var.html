Introduction

This is the fifth in a series of tests that can be used to check changes to the MRSea package do not break previously working elements. They replicate the case study in the vignette but broken down into chunks. This fifth one checks the 2D salsa functions with other variables, no 1d smooths, no panels and no interactions.
